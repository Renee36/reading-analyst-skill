# 知识网络：8 大领域 × 30 维度

## 分类框架

采用 **联合国教科文组织（UNESCO）国际教育标准分类** 与 **中国图书馆分类法（中图法）** 的交叉框架，结合个人阅读场景适配：

| 领域 | 分类依据 | 覆盖范围 |
|------|----------|----------|
| 一、自我成长与生活技能 | 中图法 B84（应用心理学）+ G4（教育）+ 生活实用类 | 面向个人效能与日常生活的知识 |
| 二、人文与文学 | 中图法 I（文学）+ UNESCO「人文学科」 | 文学作品的体裁与地域分支 |
| 三、社会科学 | UNESCO「社会科学」+ 中图法 C/D/F 大类 | 研究人类社会运行规律的学科群 |
| 四、自然科学与技术 | UNESCO「自然科学」+ 中图法 N/O/P/Q/T 大类 | 研究自然界与技术应用的学科群 |
| 五、历史与地理 | 中图法 K 大类 | 时间维度（历史）+ 空间维度（地理） |
| 六、哲学与宗教 | 中图法 B 大类（哲学/宗教） | 追问终极问题的知识体系 |
| 七、艺术与美学 | 中图法 J 大类 + UNESCO「艺术学科」 | 人类审美表达的各种形式 |
| 八、商业与职业 | 中图法 F（经济/管理）+ 职业技能类 | 面向职业发展与商业实践的知识 |

每个领域下的维度划分遵循「**互斥且穷尽**」原则（MECE），确保：每本书都能归入某个维度，人类知识版图的主要分支不被遗漏，空白区域与已有区域同等重要。

## 完整维度清单

**一、自我成长与生活技能**（8 维度）
自我管理与习惯 / 脑科学与认知 / 谈判与沟通 / 健康与身体管理 / 身心灵修行 / 个人理财与投资 / 烹饪与饮食文化 / 家居与生活美学

**二、人文与文学**（6 维度）
中国现当代文学 / 外国经典文学 / 本国特色文学（可根据读者偏好重命名，如"日本文学"） / 科幻文学 / 诗歌与散文 / 戏剧与剧本

**三、社会科学**（7 维度）
心理学 / 经济学 / 政治学与国际关系 / 法律常识 / 社会学与人类学 / 性别研究 / 教育学

**四、自然科学与技术**（6 维度）
生物与演化 / 物理与天文 / 数学 / 化学与材料 / 地球与环境科学 / 计算机与AI

**五、历史与地理**（3 维度）
世界通史 / 本国历史 / 地理与旅行

**六、哲学与宗教**（3 维度）
西方哲学 / 东方哲学 / 宗教学

**七、艺术与美学**（4 维度）
绘画与视觉艺术 / 音乐 / 建筑与设计 / 电影与影视

**八、商业与职业**（4 维度）
项目管理 / 商业与创业 / 领导力与管理 / 写作与表达

> **注**：「本国特色文学」维度可根据读者实际偏好调整名称。

## 评级标准

| 评级 | 分值 | 含义 |
|------|------|------|
| ★★★★★ | 9-10分 | 极强——体系化认知，多本交叉验证 |
| ★★★★☆ | 7-8分 | 扎实——多本涉猎且有一定深度 |
| ★★★☆☆ | 5-6分 | 有基础——有数本相关阅读 |
| ★★☆☆☆ | 2-4分 | 薄弱——仅有零星涉猎 |
| ★☆☆☆☆ | 1分 | 空白——无任何相关阅读 |

## 评估格式

| 维度 | 评级（★1-5） | 代表书目 | 评估说明 |

**排序规则**：每个领域内的维度，按评级从高到低降序排列。
